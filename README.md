# ChatApp
![alt text](https://miro.medium.com/max/1200/1*y6C4nSvy2Woe0m7bWEn4BA.png)

**About:**
This is a realtime chat app built using React.
Users are able to join group and can message others users in a realtime.

**Technology Stack:**
* React
* Firebase
* Webpack
* MObX
* Materialize 

**Running the Project**
Clone this project
* git clone < project-url.git >

Install node package manager.

Install necessary dependencies of the project.
* npm install

* npm start
